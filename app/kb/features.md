# ProcessNexus Features

## Core Features
- Real-time event streaming up to your plan limit
- REST and WebSocket APIs
- Webhook delivery with automatic retry

## Standard Plan Extras
- Custom event schemas
- 30-day data retention
- Email support (48-hour SLA)

## Enterprise Plan Extras
- Unlimited event schemas
- 1-year data retention
- Dedicated support (4-hour SLA)
- SSO and SAML 2.0
- On-premise deployment option
